# Project

This is a note-taking app/website with a to-do manager. Work still in progress

## How to run

clone and type npm install

##  Don't Forget

Don't forget to include your own Firebase config with this format:

```
React_App_API_KEY = your own key
React_App_AUTH_DOMAIN = your own key
React_App_PROJECT_ID = your own key
React_App_STORAGE_BUCKET = your own key
React_App_MESSAGING_SENDER_ID = your own key
React_App_APP_ID = your own key
```
